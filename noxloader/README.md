# NOXLOADER
**Auto Smart Universal Downloader**

Didesain khusus untuk **Termux Android**. NOXLOADER mengutamakan kebersihan UI/UX CLI tanpa output mentah dari library. Semua dibungkus rapi.

### Instalasi Termux
```bash
pkg update && pkg upgrade -y
pkg install python git -y
git clone https://github.com/YourUsername/noxloader.git
cd noxloader
python main.py
```

Bebas dari ribet. Santuy aja cok. 🤙
